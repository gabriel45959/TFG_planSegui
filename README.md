# Configuración

_Para realizar una copia en fora local hacer:_
                
+ 1-Descargar [Eclipse](https://www.eclipse.org/downloads/packages/release/luna/sr2)</br>
    + 1.a- Configurar git en eclipse hacer click [aquí](https://github.com/collab-uniba/socialcde4eclipse/wiki/How-to-import-a-GitHub-project-into-Eclipse)</br>
+ 2-Descargar [Tomcat 7 core](https://tomcat.apache.org/download-70.cgi)</br>
    + 2.a- Configurar un server en Eclipse seguir el instructivo [aquí](http://aramis-tmf.blogspot.com/2014/06/configurar-apache-tomcat-7-y-eclipse.html)</br>
+ 3- Descargar [WampServer Version 3.1.0](http://www.wampserver.com/en/) según sistema operativo</br>
+ 4- Ingresar a phpMyadmin (por defecto usuario _root_ y sin contraseña)</br>
    + 4.a- Crear una base de datos con nombre _proceso_fabricacion_tfg_</br>
    + 4.b- Importar las tablas utilizando el archivo [impactar_base_de_datos.sql](https://github.com/gabriel45959/TFG_planSegui/blob/master/plansegui/src/main/resources/img/impactar_base_de_datos.sql)</br>
+ 5- Listo!! ejecutar el proyecto con Eclipse haciendo click derecho en el proyecto>Run as>Run on Server y esperar hasta que se ejecute un explorador de internet automaticament</br>
![](https://github.com/gabriel45959/TFG_planSegui/blob/master/plansegui/src/main/resources/img/Run_proyecto_Eclipse.JPG)</br> 
    + 5.a- Ingresar al sistema utilizando como usuario admin y contraseña admin@123</br>
